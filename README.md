# دوره آموزشی Golang - کوئرا

### سرفصل‌های کلیدی دوره:
- مفاهیم پایه (متغیرها، توابع، کنترل جریان)  
- ساختارها (`struct`)، واسط‌ها (`interface`)

- کار با فایل‌ها و کتابخانه‌های استاندارد
